Expiry Date Tracker API

Што ли е „Expiry Date Tracker API“-то ?

АPI кое е изработено во Java користејќи на framework-от „Spring Boot“ и Rest API контролери, со цел да помага на екстерни веб-страници или мобилни апликации во комуницирањето со база на податоци, без притоа да се грижи за истото програмерот кој го изработува истото, во долниот дел е зададена документација за истото, при кликање на големите сини копчиња се добива информација за повиците кон API кои не интересираат.

Продукти

GET

/api/products/

Враќа листа на сите достапни продукти во базата на податоци.

Не очекува променливи.

POST

/api/products/add

Се користи за да се додаваат нови продукти во базата на податоци.

Очекува 3 Request променливи:
id (String), name (String), manufacturer_id (Integer).

DELETE

/api/products/delete/{ID}

Го бриши продуктот кој се идентификува со [ID] атрибутот.

Очекува 1 Path промелнива:
id (String)

Производители

GET

/api/manufacturers/

Враќа листа на сите достапни производители во базата на податоци.

Не очекува променливи.

POST

/api/manufacturers/add

Се користи за да се додаваат производители во базата на податоци.

Очекува 1 Request променлива:
name (String).

DELETE

/api/manufacturers/delete/{ID}

Го бриши производителот кој се идентификува со [ID] атрибутот.

Очекува 1 Path променлива:
id (Integer)

Корисници

POST

/api/users/login

Се користи за да се логираме како некој корисник, при што ако обидот е успешен ни враќа автентикациски код, кој потоа го користиме со цел да се автентицираме при креирање на барања како соодветниот корисник.

Очекува 2 Request променливи:
username (String), password (String).

POST

/api/users/register

Се користи за да се регистрира корисник во базата на податоци.

Очекува 6 Request променливи:
username (String), password (String), confirmPassword (String), name (String), surname (String), email (String).

GET

/api/users/{username}/wishlist

Се користи за да се добијат сите ставки во „wishlist“ на даден корисник, успева само доколку корисникот достави точен автентикациски код.

Очекува 1 Path променлива и 1 Request променлива:
username (String), auth_code(String).

GET

/api/users/{username}/expiryList

Се користи за да се добијат сите ставки во „expiry“ листата на даден корисник, успева само доколку корисникот достави точен автентикациски код.

Очекува 1 Path променлива и 1 Request променлива:
username (String), auth_code(String).

Продукти-Корисници

POST

/api/productsUsers/add/e

Се користи за да се додаде ставка како релацијата помеѓу Корисник и Продукт, со оваа релација се обележува истекување на даден производ, се додава само доколку доставениот автентикациски код е точен.

Очекува 4 Request променливи:
username(String), productId(String),
expiryDate(LocalDate, "YYYY-MM-DD"), auth_code(String).

DELETE

/api/productsUsers/delete/e

Се користи за да се избрише дадена ставка, само доколку автентикацискиот код кој е доставен е точен.

Очекува 2 Request променливи:
id (Integer), auth_code(String).

DELETE

/api/productsUsers/
{username}/clear/e

Ја чисти листата на ставки на даден корисник, само доколку е точен доставениот автентикациски код.

Очекува 1 Path променлива и 1 Request променлива:
username (String), auth_code(String).

POST

/api/productsUsers/add/w

Се користи за да се додаде ставка како релацијата помеѓу Корисник и Продукт, со оваа релација се обележува дека некој корисник го посакува даден производ како и колку од тој производ сака да купи во квантитет, се додава само доколку доставениот автентикациски код е точен

Очекува 4 Request променливи:
username(String), productId(String), quantity(Integer)
, auth_code(String).

DELETE

/api/productsUsers/delete/w

Се користи за да се избрише дадена ставка, само доколку автентикацискиот код кој е доставен е точен.

Очекува 3 Request променливи:
username (String), productId (String), auth_code(String).

DELETE

/api/productsUsers/
{username}/clear/w

Ја чисти листата на ставки на даден корисник, само доколку е точен доставениот автентикациски код.

Очекува 1 Path променлива и 1 Request променлива:
username (String), auth_code(String).
