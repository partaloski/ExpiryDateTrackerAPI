Expiry Date Tracker API

Products

GET

/api/products/

Retrieves a list of all products.

Expects no variables.

POST

/api/products/add

Used to add a product into the database.

Expects 3 Request Variables:
id (Integer), name (String), manufacturer_id (Integer).

DELETE

/api/products/delete/{ID}

Deletes the product which can be identified with the [ID] attribute.

Expects 1 Path Variable:
id (Integer)

Manufacturers

GET

/api/manufacturers/

Retrieves a list of all manufacturers.

Expects no variables.

POST

/api/manufacturers/add

Used to add a manufacturer into the database.

Expects 1 Request Variable:
name (String).

DELETE

/api/manufacturers/delete/{ID}

Deletes the manufacturer which can be identified with the [ID] attribute.

Expects 1 Path Variable:
id (Integer)

Users

POST

/api/users/login

Used to log in as a certain user.

Expects 2 Request Variables:
username (String), password (String).

POST

/api/users/register

Used to register a new user into the database.

Expects 6 Request Variables:
username (String), password (String), confirmPassword (String), name (String), surname (String), email (String).

GET

/api/users/{username}/wishlist

Used to get all wishlist items for a certain user.

Expects 1 Path Variable:
username (String).

GET

/api/users/{username}/expiryList

Used to get all expiry items for a certain user.

Expects 1 Path Variable:
username (String).

Products-Users

POST

/api/productsUsers/add/e

Used to add a relation User-Product, which marks a date when a product expires.

Expects 3 Request variables:
username(String), productId(Integer),
expiryDate(LocalDate, "YYYY-MM-DD").

DELETE

/api/productsUsers/delete/e

Used to delete a certain expiry from the database.

Expects 1 Request Variable:
id (Integer).

DELETE

/api/productsUsers/
{username}/clear/e

Clears the user with the username variable's expiry list.

Expects 1 Path Variable:
username (String).

POST

/api/productsUsers/add/w

UUsed to add a relation User-Product, which marks a date when a user wants to buy (wishlists) a product with a given quantity.

Expects 3 Request variables:
username(String), productId(Integer), quantity(Integer).

DELETE

/api/productsUsers/delete/w

Used to delete a certain wishlist item from the database.

Expects 2 Request Variables:
username (String), productId (Integer).

DELETE

/api/productsUsers/
{username}/clear/w

Clears the user with the username variable's wishlist.

Expects 1 Path Variable:
username (String).
